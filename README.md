# ✉️ Epmail 📧

A modern Gmail client built with ❄️ Tauri, 🍦 Vanilla TypeScript, and Bun!

## Features

- 🚀 Lightweight and fast
- 🔒 Secure email experience
- 💻 Cross-platform desktop application
- ⚡ Powered by modern web technologies
